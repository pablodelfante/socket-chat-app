# Introducción

_Repositorio de demostración para mostrar socket io como una tecnologia más incorporada, sesion básica a través de cookies, y demas_

## Como iniciar
_para iniciar el servidor local como desarrollo:<br>_
`npm run dev`


## Tecnologias usadas
* Socket.io
* Tailwind CSS
* Express
* Cookies
* ...
